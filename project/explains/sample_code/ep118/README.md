# [python: 4 ways to set namedtuple defaults (intermediate)](https://youtu.be/w4o-GT_43Yw)

```python
C2(1, 2, 3)

C2(1, 4)

C3(1)
C3(1, 2, 3)

C4(1)
C4(1, 2, 3)
```
