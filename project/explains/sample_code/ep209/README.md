# [interview question: 4 / 7 swap (beginner - intermediate)](https://youtu.be/PSXlOSJ2lZk)

Today we solve a rather silly interview question: return the other of 4 and 7 without if statements!

## Interactive examples

### Python

```python
5 and 7
5 and 7 and 9

x = 4
x == 4 and 7
x == 4 and 7 or 4

x = 7
x == 4 and 7 or 4

bin(4)
bin(7)
bin(4 ^ 0b1)
bin(4 ^ 0b11)
bin(7 ^ 0b11)
```
