# [what is sass vs scss? (beginner)](https://youtu.be/EPpjU50TYHg)

Today I talk about CSS preprocessing and the difference between sass and scss!

## Interactive examples

### Bash

```bash
virtualenv venv
. venv/bin/activate
pip install libsass

pysassc t.sass
pysassc t.sass --style compressed
pysassc t.scss
```
